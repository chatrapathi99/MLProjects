## Projects

### 1. Salary Prediction using Linear Regression
📓 [`salary_prediction_linear_regression.ipynb`](./salary_prediction_linear_regression.ipynb)

Predicts salary based on age using Linear Regression.

**What it covers:**
- Loading and exploring CSV data with Pandas
- Exploratory Data Analysis (EDA) and scatter plot visualization
- Train/Test split for proper model evaluation
- Linear Regression model training using Scikit-learn
- Model evaluation using R² Score and Mean Absolute Error (MAE)
- Visualizing regression line against actual data points

**Tech Stack:** Python | Pandas | NumPy | Scikit-learn | Matplotlib  
**Dataset:** `age_salary.csv`  
**Key Result:** Model explains X% of salary variation using age as predictor
